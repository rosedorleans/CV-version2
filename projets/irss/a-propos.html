A Propos

Stage de deux mois dans l'entreprise IRSS.
Du 10 mai au 2 juillet 2021

Projet : mise à jour et refonte du site IRSS.fr
- Langages : PHP / MySQL / HTML / CSS / Javascript, et CMS Wordpress
- Optimisation des sites Web (référencement, UX)
- Rédaction des documentations techniques et fonctionnelles

"J'ai créé une carte interactive au format SVG pour que les utilisateurs puissent cliquer sur les villes de France où se situent les écoles. Un fichier Javascript permet de :
- changer la couleur des villes dans le texte et sur la carte au passage de la souris.
- afficher un modal au clic de la souris (exemple visible sur Caen)

J'ai développé un calendrier répertoriant tous les événements des écoles IRSS et j'y ai ajouté une page pour filtrer les événements par type de formation, ville et type d'événement"

Le calendrier n'est pas présent sur le site à ce jour. Un plugin The Event Calendar a été choisi.
Ce choix a été fait pour garantir la bonne mise à jour des composants.
La carte interactive est présente sur la page d'accueil du site.
